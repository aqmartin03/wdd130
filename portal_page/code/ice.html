ICE Challenges

Here is a list of every ICE challenge that I have done in my Web Fundamentals class. Every project has a description and a link. Click on a link below, and have fun looking at my work!

Prophet Cards: in this challenge, I learned the basics of HTML and CSS. I learned how to move things to the center of the screen, how to implement colors to each element, and to create a certain wireframe.

Prophet Cards

Apostle Spotlight: this challenge was similar to the first one, but a bit more complex and advanced. I used a grid to position the pictures of apostles in an orderly manner. I learned how to grid and how to create gaps that made the design look professional and neat. I also learned how to put objects in front of a background picture.

Apostle Spotlight

Favorite Devotional: in this one, I learned how to create a form and place it in a table. I put some information inside the table, and I added an extra row so that someone could put some input in if they wished.

Favorite Devotional

Father's Day: in this challenge, I experimented with CSS without affecting the already existing HTML. I was given the opportunity to be creative, which I appreciated because it allowed me the chance to prove that I have been learning how to affect a website design. I changed coloring, formatting, and even text sizing on this design.

Father's Day

Flags: in here, I received additional practice with using CSS. This time, the main focus was on gridding. I learned how to determine the amount of rows and columns in each object through this example.

Flags

Positioning: this is one of the more difficult challenges that I had the opportunity to attempt. I learned some advanced grid and positioning strategies in CSS. I also learned more about how to overlap objects. While this was a difficult one, I had fun practicing!

Positioning

Gallery: this was a unique one. I learned how to use a photo editor and optimize photos for a design. Before, when loading the website design, the photos had taken a long time to populate on the screen. By optimizing the photos, and converting them to different file types, I learned how to make them load on the screen more efficiently.

Gallery

Signup: everything came together in this final ICE challenge. I changed the HTML and CSS to create a design that consists of a form, images, and rounding the corners of the box. I learned that patience means everything, especially when it feels like things aren't working that you want them to. This took a while, but eventually, I was able to create the intended outcome.

Signup
